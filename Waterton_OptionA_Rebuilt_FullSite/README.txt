Open index.html in your browser.

Admin controls:
- Toggle seasonal waitlist: site-config.js → SITE_CONFIG.seasonal.status ('available' or 'full')
- Mark a seasonal site available: site-config.js → SITE_CONFIG.seasonal.sites.A14 = true
- Daily calendar availability: site-config.js → SITE_CONFIG.daily.calendar['YYYY-MM-DD'] = { availableSites: 0, sites: [] }
